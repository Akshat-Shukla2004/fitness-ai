# 🏋️ Fitness-AI

A machine learning-based project to predict calories burned and provide basic diet and workout recommendations.

---

## 🚀 Features

* Calorie prediction using ML
* Basic diet recommendation system
* Workout suggestions
* Data preprocessing and cleaning

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
